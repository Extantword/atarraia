Este repositorio está organizado por carpetas. La idea es que cada carpeta contiene su propio proyecto standalone. Lo más común es que
cada uno de estos proyectos es las siguientes de la dos opciones:

- Un archivo .ipynb, el cual se ejecuta desde Google Colab
- Una página web

La idea es que este repositorio es una página web desde la cual se puede acceder a los distintos proyectos.
